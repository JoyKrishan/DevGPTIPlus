### Using `operate` Modes

#### Voice Mode
- Install the additional `requirements-audio.txt`
```
pip install -r requirements-audio.txt
```
**Install device requirements**
- For mac users:
```
brew install portaudio
```
- For Linux users:
```
sudo apt install portaudio19-dev python3-pyaudio
```
Run with voice mode
```
operate --voice

```
